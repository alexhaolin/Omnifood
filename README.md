Omnifood Web Design

Goals of the project:
1. Design website and develop front-end web step-by-step by building a beautiful website from complete scratch, using the latest web technologies HTML5, CSS3 and some jQuery.
2. Apply Responsive Web Design with media queries for multiple resolutions and browsers.
3. Use JavaScript and jQuery to add three dynamic animation effects
	-  Sticky Navigation
	-  Responsive navigation
	-  Scrolling to elements from the navigation and buttons
4. Magnific Popup to show details of elements
5. Flex box to lay out elements in a 1-dimensional row
6. Use PHP to send contact form data to the server side
7. Performance optimization
8. Search engine optimization
9. Lunch the website
	-  Choose and buy a domain name
	-  Buy web hosting
	-  Upload my website
10. Use Google Analytics for maintenance
